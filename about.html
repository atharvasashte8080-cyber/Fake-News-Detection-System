Project Overview

The Fake News Detection System is an intelligent software solution designed to identify misinformation in digital news articles. By leveraging Natural Language Processing (NLP) and Machine Learning (ML), the system analyzes linguistic patterns and calculates the probability of an article being deceptive.

Technical Architecture

This project uses a decoupled architecture where the Frontend handles user interaction and the Backend performs heavy text computation.
